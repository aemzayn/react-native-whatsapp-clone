# What is dis?

React Native (expo) WhatsApp clone for learning purposes

## Clone

```
git clone https://github.com/aemzayn/react-native-whatsapp-clone
```

## Install

```
cd react-native-whatsapp-clone && npm install
```

## Run

```
npm run android
```
